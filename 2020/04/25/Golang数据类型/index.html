Golang数据类型

持续更新

整数:

int 有符号，大小随计算机位数变化(64位操作系统int类型是64位的,32同理)

以下是有符号的整数类型

  • rune
  • int8
  • int16
  • int32
  • int64
  • rune = int32

以下是无符号的整数类型

  • byte
  • uint8
  • uint16
  • uint32
  • uint64
  • byte = uint8

这些类型，无法直接相互赋值或操作

字符串:

string
字符串不可变
s[0] = ‘c’ <- 这个是不可行的
布尔: bool
